# Парсер политических партий

Программа для парсинга списка политических партий с сайта Министерства юстиции РФ.

 Описание

Скрипт извлекает информацию о политических партиях с официального сайта Минюста:
- Наименование партии
- Ссылки на документы (устав, регистрационные сведения)

 Подготовка данных

HTML-страница скачивается автоматически при запуске скрипта с адреса:
https://minjust.gov.ru/ru/pages/politicheskie-partii/
